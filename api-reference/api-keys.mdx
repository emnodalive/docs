---
title: "API keys"
---

To make every API call, you'll need your key. Always include it in the request header under `x-api-key`.

<Tip>
  You can find your sandbox and production API keys under the **Integration** tab in your Hub.
</Tip>

**Request example:**

```
curl --request POST \
  --url https://api.stage.rocka.live/api/payments/logo \
  --header 'Accept: application/json, text/json, text/plain' \
  --header 'Content-Type: application/*+json' \
  --header 'x-api-key: your-api-key-goes-here' \
  --data '{
  "currency": "EUR"
}'
```
