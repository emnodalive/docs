Our API is designed to help you smoothly process payments and provide your customers with the best user experience.

### API environments

| Environment | URL                                                          |
| :---------- | :----------------------------------------------------------- |
| Sandbox     | [https://api.stage.rocka.live](https://api.stage.rocka.live) |
| Live        | [https://api.rocka.live](https://api.rocka.live)             |

### API endpoints

| Request method | Endpoint                                                                     |
| :------------- | :--------------------------------------------------------------------------- |
| POST           | [Display customised payment buttons](ref:display-customised-payment-buttons) |
| POST           | [Create a payment](ref:payment_paymenturl)                                   |
| GET            | [Check payment status](ref:payment_get)                                      |
| GET            | [Get the list of top banks](ref:get_providers_top)                           |
| POST           | [Create payment links](ref:create-payment-links)                             |
| POST           | [Create a payout](ref:create-a-payout)                                       |
| GET            | [Check payout status](ref:check-payout-status)                               |
| GET            | [Export payments](ref:payments_getall)                                       |
| GET            | [Export payouts](ref:export-payouts)                                         |
| POST           | [Create a payment](ref:create-a-card-payment) (Card payments)                |
| GET            | [Check payment status](ref:check-card-payment-status) (Card payments)        |
| POST           | [Create a payout](ref:create-a-card-payout) (Card payouts)                   |
| GET            | [Check payout status](ref:check-card-payout-status) (Card payouts)           |
